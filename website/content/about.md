StrictDoc is a Python-based tool for technical documentation and requirements management.

- **Open source**, Apache 2.0 license
- **Started in 2019**
- **Installable via pip**, Python only

It was built by engineers working on critical software, who wanted requirements management that fits naturally into a developer workflow: plain text, Git, and the command line.
