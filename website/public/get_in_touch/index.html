Get in touch

All StrictDoc features are driven by user feedback. Some users are active contributors to the project.

Documentation

The full documentation is at strictdoc.readthedocs.io, covering installation, configuration, SDoc format reference, and guides for all major workflows.

GitHub

Bug reports, feature requests, and contributions all go through the GitHub repository.

For questions and longer discussions, use GitHub Discussions.

Discord

For real-time chat, join the StrictDoc Discord channel.

Mailing list

For announcements and broader discussions: strictdoc.groups.io

Office hours

Weekly open video call, every Tuesday at 17:00 CET. Drop in to ask questions, share how you use StrictDoc, or follow ongoing development.
